# 수당헬프(SudangHelp) 개발 기술 기록

수당헬프는 **환율·여행 금융 계산기**를 중심으로  
일상에서 실제로 쓰이는 금융 계산 도구를 만드는  
웹 기반 계산기 서비스입니다.

> A practical FinTech project focused on real-world exchange calculators  
> and offline-first PWA experiences.

---

## 🔍 무엇을 만드는가

수당헬프는 단순 정보 제공 페이지가 아닌,  
**실제 사용을 전제로 설계된 계산기형 웹 애플리케이션**을 개발합니다.

현재 핵심 서비스는 다음과 같습니다.

- 🌍 해외여행 환율 계산기 (49개국 지원)
- 💳 카드 수수료·환전 수수료 포함 실결제금액 계산
- 📱 앱 설치 없이 바로 실행되는 PWA 계산기
- ✈️ 비행기 모드에서도 작동하는 오프라인 계산

👉 서비스 바로가기  
https://sudanghelp.co.kr/travel/exchange-calculator/

---

## 🤔 왜 이 프로젝트를 만들었나

해외여행 중 환율 계산은 생각보다 번거롭습니다.

- 여전히 **0 떼고 나누기** 같은 방식이 쓰이고
- 대부분의 계산기는 **온라인 연결이 필수**
- 카드 수수료, 실제 결제 금액은 따로 계산해야 함

수당헬프는 이런 불편을 해결하기 위해  
**“지금 이 상황에서 바로 쓸 수 있는 계산기”**를 목표로 만들어졌습니다.

---

## 🛠️ 기술적 접근 방식

이 프로젝트는 콘텐츠가 아니라 **소프트웨어**로 설계되었습니다.

- **PWA (Progressive Web App)**
  - Service Worker 기반 캐싱
  - 네트워크 불안정/오프라인 환경 대응
- **JavaScript 계산 로직**
  - 실시간 환율 반영
  - 수수료 포함 실청구액 계산
- **SEO & 퍼포먼스**
  - 구조화 데이터(JSON-LD)
  - Core Web Vitals 최적화
  - JS 기반 다국어/다통화 랜딩 구조

---

## 📐 개발 원칙

수당헬프의 모든 계산기는 다음 기준으로 설계됩니다.

- ✔ 앱 설치 없이 즉시 사용 가능
- ✔ 실제 조건(수수료·이자·기간)을 숨기지 않음
- ✔ 모바일·여행 환경 우선 설계
- ✔ 계산 결과와 로직을 투명하게 공개
