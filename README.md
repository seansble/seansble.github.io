## 🔗 프로젝트 성격

- 개인/독립 프로젝트
- 광고용 랜딩이 아닌 **실사용 서비스**
- 계산기 로직·구조·설계는 직접 구현 및 운영

---

## 📌 앞으로의 방향

1. 환율 계산기 고도화 및 안정화
2. 여행 금융 → 대출·이자 계산기로 확장
3. "검색해서 들어와 바로 쓰는 계산기"가 아닌, **신뢰 가능한 생활 금융 도구**로 성장

- **Features**: PWA, Offline Support, Multi-currency Calculator
